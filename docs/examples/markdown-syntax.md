# Markdown 语法参考

完整的 Markdown 语法指南。

## 标题

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

## 强调

```markdown
*斜体* 或 _斜体_
**粗体** 或 __粗体__
***粗斜体*** 或 ___粗斜体___
~~删除线~~
```

## 列表

### 无序列表

```markdown
- 项目一
- 项目二
  - 子项目
  - 子项目
- 项目三
```

### 有序列表

```markdown
1. 第一项
2. 第二项
3. 第三项
```

## 链接和图片

```markdown
[链接文本](https://example.com)
![图片描述](图片路径)
```

## 引用

```markdown
> 这是一个引用
> 可以多行
```

## 代码

### 行内代码

```markdown
`行内代码`
```

### 代码块

````markdown
```语言
代码内容
```
````

## 表格

```markdown
| 表头1 | 表头2 | 表头3 |
|-------|-------|-------|
| 单元格 | 单元格 | 单元格 |
| 单元格 | 单元格 | 单元格 |
```

## 水平线

```markdown
---
```

## 任务列表

```markdown
- [x] 已完成
- [ ] 未完成
- [ ] 另一个任务
```

## 转义字符

使用 `\` 转义特殊字符：

```markdown
\*不是斜体\*
```
